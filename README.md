# BnMindQA
BnMindQA is a Retrieval-Augmented Question Answering (RAG-QA) system for Bangla mental health support. It combines hybrid retrieval (BGE-M3 + BM25 + RRF + Cross-Encoder Reranking) with a QLoRA fine-tuned Gemma-3-4B model to generate accurate, context-grounded Bangla responses from curated mental health resources.
